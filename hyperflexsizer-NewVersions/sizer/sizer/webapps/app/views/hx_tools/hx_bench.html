HX Bench

Cisco HxBench is a storage performance testing tool, which can be used to measure the storage infrastructure performance using defined test scenarios and workloads. HxBench uses the industry standard Vdbench storage performance benchmarking tool. HxBench is provided as a virtual appliance to deploy and run the tests. HxBench simplifies performance benchmarking for virtualized environments.
