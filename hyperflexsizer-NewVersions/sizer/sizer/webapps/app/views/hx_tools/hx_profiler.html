HX Profiler

Hx Workload Profiler is vCenter characterization tool that used to estimate the compute, storage and network usage of hosts and VMs across multiple vCenter deployments. The output of the tool can be used to feed into HxSizer to size the HyperFlex cluster required to fulfil the workload requirements as estimated by the Profiler.
